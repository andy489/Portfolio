    Building Suffix Automaton

    A suffix automaton is a minimal finite automaton that recognizes all suffixes of a string. Built online by adding characters sequentially, it maintains equivalence classes of end positions using suffix links. This efficient O(n) algorithm compresses all substrings into a compact directed acyclic graph structure, enabling fast substring queries and pattern matching.
